# Harness Reference

Supporting detail extracted from CLAUDE.md to keep the agent instruction file concise.

## ACTIVE_TASK.md — Empty Schema Template

When creating `ACTIVE_TASK.md` for a new project, use this exact template:

```markdown
# Active Task
<!-- Status: idle -->

## Requirement
<!-- task writes here -->

## Design
<!-- design writes here -->

## ADRs
<!-- grill writes here -->

## Risks
<!-- risk writes here -->

## Implementation Log
<!-- code, tdd, refactor write here -->

## Test Results
<!-- tests, coverage, verify write here -->

## Review Findings
<!-- review, audit write here -->

## Deploy Checklist
<!-- deploy writes here -->

## Post-Deploy
<!-- ship writes here -->
```

The `<!-- Status: idle -->` sentinel distinguishes an initialized-but-empty file from an uninitialized one. `close` resets to this template after each archive.
